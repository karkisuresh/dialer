Dota Underlords

Introduction

Dota Underlords, more commonly known as just Underlords, is a tactical strategy game developed by Valve which is available for free at Steam online store. Dota Underlords is based on the a very popular mod of Dota2 called Autochess. Underlords is a chess-like game with a twist where the pieces fight automatically when placed on the board. The pieces are characters from Dota2 called heroes with their unique abilities. The game also contains items that provides special stat boost and unique abilities. The characters can be leveled up by combining three of same pieces where the maximum level is 3. A total of 10 heroes can be placed on the board and the team with better synergy is most likely to win the game. The game is also available for android and IOS.

Game Modes

The game provides three game modes to the user which are tutorial mode, solo/bot mode and multiplayer mode.

Tutorial Mode

In Tutorial Game mode, a comprehensive tutorial is provided to the beginners so that they can get started with the game. Tutorial involves playing games with bot where clear instructions are provided to the user.

Solo/Bot Mode

Solo/Bot is a single player game mode where the user plays with 7 other bots. This is an actual game where player can practice against bots.

Multiplayer Mode

In Multiplayer Game mode, the user plays and competes with 7 other human players. This is also called ranked matchmaking as the user is assigned a rank or medal based on the performance.

Ranking System

On playing multiplayer games, players are given certain ranks. Players are able to move up the ranks by securing a top four finish, they lose rank when they finish in fifth place or lower. The ranks/medals in the game along with their tiers is given in the table below.

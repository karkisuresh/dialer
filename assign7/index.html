About

For this assignment, I created a conversational bot interface of smart light application which is able to control the light and includes features like knowing the state of light of a room, turning it on or off, change its color, brighten or dim the light and schedule a light to turn on or off at particular time.

For this assignment, the bot interface was tested multiple times by a user, the list of changes made after each test considering the results and feedbacks from the user is given below:

Test 1

  • The application was not able to detect simple command like "turn on the light" instead the user had to say "turn the light on". So, I had to include necessary training for it to work properly.
  • The application was able to detect rooms well from the commands except for one command of "Can you turn on the bathroom's light?" where the application was detecting can as a room. So, I had to include necessary training to handle that.
  • Also, from observing the user interaction with the bot, it was found that user like to use short commands such as "light on kitchen" or "bathroom color red" instead of saying the whole sentence and for some cases the application wasn't able to give the intended result. So, I had to include necessary training to make sure that the correct intent was detected.

Test 2

  • Along with changing the color of light to a specific color, the user mentioned a request to set multiple fun colors like that of a disco bulb. So I added a disco mode and normal mode which user can activate using command such as "disco mode" or "normal mode".
  • As per the feedback of user, the increase brightness and decrease brightness has been changed to just dim and brighten, so instead of just constantly increasing and decreasing brightness, the user can just say dim or enlighten. Dim and enlighten will have predefined brightness level.
  • I have added a feature where user is able to schedule the light to turn on or off at certain time or date or after a certain duration as user mentioned it as a nice to have feature. So, user is now able to schedule through commands such as "turn off bedroom's light after 30 min".

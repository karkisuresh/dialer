About this Project

For this assignment, we used magnifier to navigate through a website and perform an action in High Contrast Mode. I went to amazon website and performed an action of buying something. The things I learned and challenges faced while using magnifier on High contrast mode are as follows:

  • It was difficult distinguishing the background and a window on High Contrast Mode.
  • Similarly, it was difficult to separate between buttons.
  • While using magnifier, there was too little information than it would have been on normal zoom.
  • User needs to scroll more.
  • It was diffcult and annoying at times to distinguish when to slide mouse to see more and when to scroll.

  • High Contrast Mode
